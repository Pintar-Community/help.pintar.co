Acara Hari Ini

Pada bagian ini berisi Acara Hari Ini yang berupa Live Session. Pelaksanaan Live Session yang akan muncul pada Acara Hari ini adalah Live Session yang akan dilaksanakan pada saat Instruktur mengakses LMS.

Cara mengakses Live Session dari menu Acara Hari Ini :â€‹

  1. Untuk menampilkan secara keseluruhan, Anda dapat klik button Lihat Semua pada pojok kanan atas, kemudian akan muncul tampilan seperti gambar.

  1. Untuk membuka tampilan Live Sessionnya, Anda dapat klik judul Live Sessionnya, maka akan diarahkan pada Live Session pada kelas tersebut.

  1. Anda dapat join Live Session dengan cara klik button Masuk. Seluruh user sudah dapat join Live Session 5 menit sebelum Live Session berlangsung.
